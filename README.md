# 🏓 Breakout Web

A classic Breakout / Arkanoid clone built with HTML5 Canvas and vanilla JavaScript.

## Features
- 6 brick rows with color-coded HP values
- Ball physics with angle-based paddle deflection
- Power-ups: Wide Paddle, Multi-ball, Slow Ball, Extra Life
- Progressive level system — bricks reform each level
- Score multiplier for combos
- Local best score (localStorage)
- Mobile touch controls

## Controls
| Input | Action |
|-------|--------|
| ← → / Mouse | Move paddle |
| Space | Launch ball |
| Touch drag | Move paddle |

## Tech Stack
- HTML5 Canvas
- Vanilla JavaScript (ES6+)
- CSS3
